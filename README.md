# MediaConnect 🎥✨

A complete web application that allows users to **log in, sign up, save contacts, manage profile, and play/store videos using LocalStorage**.  
This project demonstrates how to build a lightweight, offline‑friendly app without external databases.

## 🚀 Features
- 🔑 **User Authentication** – Login & Signup system with validation.
- 📇 **Contact Manager** – Add, view, and delete contacts.
- 👤 **Profile Page** – User info with saved data.
- 🎥 **Video Player** – Upload, save, and play videos dynamically.
- 💾 **Offline Persistence** – Powered by LocalStorage & SessionStorage.

---

## 🛠️ Tech Stack
- **HTML5 / CSS3 / JavaScript**
- **LocalStorage & SessionStorage APIs**
- **FontAwesome & Animate.css** for icons & animations
